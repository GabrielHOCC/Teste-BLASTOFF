# QUESTÃO 11:

 Na função JavaScript a seguir, responda quais serão os resultados respectivos: 

~~~ JavaScript
function multiplicacao(num1, num2) {
    let resultado = num1 * num2;
    return resultado;
}
multiplicacao(4, 7);
multiplicacao(5, 6);
multiplicacao(3, 3);
~~~

## Resposta:

### **Os resultados são respectivamente: 28, 30, 9**
